# 🚀 AssetFlow

> Enterprise Asset & Resource Management System

---

## 📌 Overview

AssetFlow is an enterprise-grade web application that helps organizations manage assets, employees, departments, maintenance requests, resource bookings, audits, transfers and analytics from one centralized dashboard.

This project is being developed during a Hackathon using a modern React frontend and Supabase backend.
